Description:

AWK is a programming language that can be used script operations such as parsing, editing, and extract table column values.

Infer Experiment calculates the fractions of forward and inversely mapped reads onto genomic features with known orientation.

HTSeq-count calculates the number of reads that have been mapped against the genomic feature of a reference genome. It is very similar to its alternative featureCounts: both require the provided input alignment files to be in BAM or SAM format, as well as a genome feature file in GTF or GFF format.
Although these tools return similar results, counts can vary considerably in case of paired-ended data. FeatureCounts is faster than its counterpart HTSeq-count, and its count table takes into account feature overlaps. On the contrary, HTSeq-count is slower and more consirvative in terms of counting fragments whose alignment overlaps multiple features. Compared to featureCounts, HTSeq-count is older, and therefore widely used in the community.

DESeq2 assesses the level of gene expression. This tool computes ranking and visualization of count data for hypothesis testing under multiple sampling conditions.
