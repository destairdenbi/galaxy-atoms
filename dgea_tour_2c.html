Description:

FastQC is one of the preferred tools for quality control. This tool analyses FASTQ files, and produces graphs to allow fast inspection of sequencing data sets.

Trimmomatic offers a pipeline-based architecture to perform various trimming/filtering steps on paired-end and single-end illumina data sets. This tool offers an easy to use interface to apply each step in the order specified by the user.
